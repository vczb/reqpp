# Request Plus Plus

Is a tinny and light request library

## Install

Install using a [npm package](https://www.npmjs.com/package/reqpp).

```bash
  //with npm
  npm install reqpp

  //with yarn
  yarn add reqpp
```

## Features

- abort requests
- retry requests
